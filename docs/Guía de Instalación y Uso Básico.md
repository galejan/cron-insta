# Guía de Instalación y Uso Básico de Cronista

## 1. Instalar Git (Opcional pero recomendado)
Cronista integra Git de forma transparente para proteger tu trabajo creando un checkpoint automático cada vez que cierras la aplicación 
- **Windows:** Descarga e instala desde git-scm.com o desde el instalable de esta carpeta. Solo dale a siguiente siguiente y deja todo por defecto, es una aplicacion que usamos todos los programadores del mundo... es segura e inofensiva


## 2. Instalar Cronista
1. Haz doble clic sobre el archivo `.msi` que se te ha compartido. Aunque te salga un aviso de que el programa puede ser peligroso, eso sale porque como es un proyecto en desarrollo, no esta aun registrado con un certificado de windows valido.
2. Sigue las instrucciones en pantalla del asistente de instalación de Windows.
3. Abre la aplicación **Cronista**. 
*(Si instalas Git más adelante, el historial comenzará a registrarse desde ese momento [3]).*

## 3. Importar el proyecto de ejemplo
Cronista es un editor **Local-First**, lo que significa que todo se guarda en local y un proyecto es simplemente una carpeta con archivos Markdown (`.md`) para el texto y `.json` para los metadatos e índices 
2. Dentro de Cronista, dirígete al panel inferior izquierdo y haz clic en el botón **"📁 Abrir proyecto"** (o usa el atajo `Ctrl + O`) 
3. Selecciona la carpeta de tu proyecto de ejemplo en el explorador de archivos.
4. Cronista cargará instantáneamente los capítulos, personajes y la línea de tiempo.

## 4. Crear un nuevo proyecto.
1. Dale a nuevo proyecto y elige en primer lugar la carpeta donde se creara tu proyecto (normalmente Documentos)
2. Ahora ponle nombre a tu proyecto. Esto creara una nueva carpeta donde se inicializaran las opciones del proyecto y donde se guardaran tus notas, personajes, cronograma, asi como el control de versiones si instalaste GIT. 
En este caso te pedira unos datos, puedes ponerlos o dejarlo con los datos precargados.
3. Dale nombre a tu primer capitulo y comienza tu proyecto.

## Atajos de Teclado
Para agilizar tu flujo de trabajo en el editor, Cronista cuenta con los siguientes atajos:

| Atajo | Acción |
| :--- | :--- |
| `Ctrl + Shift + ←` | Colapsar panel lateral |
| `Ctrl + Shift + →` | Maximizar panel lateral |
| `Ctrl + ←` / `→` | Reducir / ampliar panel lateral (5 %) |
| `Ctrl + P` | Mostrar / ocultar panel de herramientas |
| `Ctrl + S` | Guardar ahora |
| `Ctrl + N` | Nuevo capítulo |
| `Ctrl + O` | Abrir otro proyecto (cierra el actual) |
| `Ctrl + Shift + N` | Nuevo proyecto (cierra el actual) |
| `Ctrl + ↑` / `↓` | Subir / bajar nivel de título |
| `Ctrl + D` | Insertar guion de diálogo ( — ) |
| `Ctrl + +` / `-` | Aumentar / reducir tamaño de letra |
| `Ctrl + Enter` | Si se pulsa sobre un personaje lo fija al editor de texto
| `Ctrl + I` | Importar proyecto
| `Ctrl + T` | Navegar entre pestañas Capitulos, personajes, notas.
| `F11` | Pantalla completa |
| `F1` o `?` | Mostrar / ocultar panel de ayuda |

| Puedes mirar la ayuda para ver los atajos de teclado actualizados.
